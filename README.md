#  Stock Analysis Tool

## Overview
This project is an interactive stock analysis tool built using Python.  
It helps users analyse stock prices, returns, and risk.

## Features
- Stock price visualization
- Moving averages (MA20, MA50)
- Return and volatility analysis
- Interactive stock selection

## Data Source
Data is obtained from Yahoo Finance using the yfinance library.

## How to Run
1. Open the notebook (.ipynb file)
2. Run all cells
3. Use the interactive widget to select stocks
